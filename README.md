"# WeatherApp" 

A simplistic full screen weather app which obtains current weather information and a 7 day forecast from the National Weather Service and displays it. 

***Can only be used in the United States***

To display weather for your area, change latitude and longitude constants depicted below to your location & recompile. 

        // Your coordinates
        private static readonly double Lat = 45.688867;
        private static readonly double Lon = -122.651783;
